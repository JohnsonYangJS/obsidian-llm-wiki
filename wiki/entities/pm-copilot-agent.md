## PM Copilot 概述

PM Copilot 是作者为自己打造的个人 AI 产品经理助手，是一个将个人知识、方法论与 AI 能力深度整合的工具产品。

## 开发背景与动机

作者作为产品负责人/数字化负责人，在日常工作中积累了大量的方法论、书籍笔记和项目决策记录。这些知识散落在脑海、电脑和网盘中，每次使用都需要手动整理后交给 AI 处理。为了解决这一痛点，作者决定打造一个统一的 AI 知识库，让 AI 能够理解并主动调用这些知识。

## 核心功能

PM Copilot 的目标是一个完整的 PM OS（产品经理操作系统），集成以下核心功能：

- **知识管理**：整合个人读书笔记、方法论、项目决策
- **需求管理**：AI 辅助需求梳理与分析
- **竞品分析**：结构化的竞品调研框架
- **路线图规划**：数据驱动的产品路线图制定
- **原型设计**：结合 AI 的快速原型生成
- **代码实现**：以前端为主的代码辅助生成

## 技术架构

- **开发框架**：基于 MyAgents 开源项目，使用 Tauri 构建桌面端
- **AI 主力组合**：Claude Code + 智谱AI（GLM）的 Coding Plan API
- **开发理念**：Vibe Coding —— 在使用 AI 辅助编程的过程中享受创造的乐趣

## AI 工具演进历程

作者从 2022 年底 ChatGPT 发布开始使用 AI 工具，经历了三次主力工具迁移：

1. **ChatGPT**（最久使用）：文档、图片、PPT、邮件等日常辅助
2. **Gemini**：短暂使用，未形成不可替代的理由
3. **Claude Code + 智谱AI**：当前主力组合，覆盖日常产品和数据工作需求

## 核心理念

作者强调，这个产品的起点不是「做一个产品」，而是源自一个朴素的需求——为自己打造趁手的工具。过程本身就是学习，作为一个非程序员，用 Claude Code 去构建 AI 产品带来的获得感远超单纯使用工具。
